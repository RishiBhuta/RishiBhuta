## 🚀 Projects I'm Proud Of

### 🏦 [LoanIQ — AI-Powered Loan Eligibility Engine](https://github.com/RishiBhuta/LoanIQ)
ML model trained on **614 real loan applications** that predicts eligibility and explains every decision using **SHAP (SHapley Additive exPlanations)**. No black boxes — users see exactly which factors helped or hurt their application. Built for transparency, grounded in real lending data.

### ⚖️ [LegalEase India — AI Legal Awareness Platform](https://github.com/RishiBhuta/LegalEase-India)
AI-powered platform making Indian law accessible to ordinary citizens. Ask legal questions, get answers grounded in Indian law, browse live legal news, and access key legal documents — all in one place.  
`FastAPI` · `React` · `Google Gemini`

### 🎰 [Roulette Royale — Premium Casino Simulation](https://github.com/RishiBhuta/Roulette-Royale)
A web-based roulette experience built to feel genuinely premium. Dark-and-gold aesthetic, professional-grade animations, realistic betting engine. Because if you're going to build a game, build it right.  
`React` · `Vite`

### 🚗 [Vehicle Insurance Prediction](https://github.com/RishiBhuta/Vehicle-Insurance-prediction-)
End-to-end ML pipeline for vehicle insurance cross-sell targeting — from raw data to deployment-ready inference.  
`Python` · `scikit-learn` · `pandas`

### 🎯 [AttendEase — Attendance Calculator](https://github.com/RishiBhuta/AttendEase)
A lean, no-nonsense C-based console tool that tells students exactly how many classes they need to attend to stay above the 75% threshold. Simple problem. Clean solution.  
`C`
